# [0504. 七进制数](https://leetcode.cn/problems/base-7/)

- 标签：数学
- 难度：简单

## 题目大意

给定一个整数 num，将其转换为 7 进制数，并以字符串形式输出。

## 解题思路

对 num 不断取余整除，然后将取到的余数进行拼接成字符串即可。

## 代码

```Python
class Solution:
    def convertToBase7(self, num: int) -> str:
        if num == 0:
            return "0"
        if num < 0:
            return "-" + self.convertToBase7(-num)
        ans = ""
        while num:
            ans = str(num % 7) + ans
            num //= 7
        return ans
```
